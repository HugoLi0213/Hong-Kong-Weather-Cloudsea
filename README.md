# 香港天氣和雲海

本專案為一個現代化的香港天氣與雲海預測網站，支援手機與桌面瀏覽，整合 Open-Meteo 及香港天文台 API，並提供 18 區天氣、雲海預測、空氣質素等功能。

## 主要功能
- 18區天氣（香港天文台 API）
- 天氣總覽（Open-Meteo API）
- 雲海預測（Open-Meteo API，支援逆溫層查詢教學）
- 空氣質素指數（AQHI）
- 行動裝置友善設計

1. `npm install`
2. `npm run dev`
3. 於瀏覽器開啟 http://localhost:5173
